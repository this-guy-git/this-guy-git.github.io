Core Commands

print

Outputs text or variables. Syntax: print"str" or print variable.

print"Hello World"  { No space needed }
print x, y  { Adds space between }
print"Value:"x  { No space added }

input

Captures user input. Stores or prints directly.

name = input  { Stores input }
printinput  { Prints input directly }

import

Imports variables from other Guython files.

importutils.gy
print utils.version

alias

Creates command shortcuts.

alias print = p
p"This works!"

Control Flow

if/else

Conditional execution with dot indentation.

ifx,>,10.
.print"Big"

else
.print"Small"

while

Loop while condition is true.

x = 0
whilex,<,5.
.print x
.x = x+1

goto

Jumps to specific line number.

print"Line 1"
goto3
print"Line 2"  { Skipped }
print"Line 3"

eval

Executes dynamic code.

code = "5 + 5"
result = eval code
print result  { 10 }

File Operations

read

Reads files with modifiers.

read "/path/to/file.txt" -lines
read "config.gy" -ign

write

Writes to files with modifiers.

write "log.txt" "New entry" -add
write "newfile.txt" "Content" -create

ngi

New Guython instance.

ngi -s "print'Running in new instance'"

guython

Runs Guython scripts.

guython script.gy

Package Management (GPD)

gpd pkgs

Lists all available packages.

gpd pkgs

gpd install

Installs packages.

gpd install requests

gpd import

Imports installed packages.

gpd import requests

gpd update

Updates packages.

gpd update requests
