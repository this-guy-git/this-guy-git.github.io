Resistance Measurement

What Is Resistance?

Resistance is how much a component or wire resists the flow of electricity. It tells you how hard it is for current to pass through a part of the circuit.

How It’s Used

  • Checks if resistors are the correct value.
  • Measures how much a wire or component slows down current.
  • Helps understand why circuits behave the way they do.

Testing

Upload this code to test the Ohmeter. 
#include <LiquidCrystal.h>

LiquidCrystal lcd(7, 6, 5, 4, 3, 2);
const float R1 = 100.0;

void setup() {
  lcd.begin(16, 2);
}

void loop() {
  long sum = 0;
  for(int i=0;i<10;i++) sum += analogRead(A5);
  int rRaw = sum / 10;
  float R2 = ((R1 * (1023.0 / rRaw - 1.0) - 15.0) / 1.1437);

  lcd.clear();
  lcd.setCursor(0,0);
  lcd.print("RESISTANCE:");
  lcd.setCursor(0,1);
  lcd.print((int)R2 + " Ohms");
  delay(1000);
}
Then you can touch the leads to any resistor up to 10k Ohms, the value should show on the LCD.
