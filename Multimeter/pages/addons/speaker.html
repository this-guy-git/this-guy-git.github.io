Speaker

In real multimeters, they have a speaker that plays when the circuit has continuity. This addon adds a speaker that does the same thing.

Click on the Image to zoom in.

Change the continuity function to this: 
void continuity()
{
  int connected = digitalRead(9);

  lcd.clear();
  lcd.setCursor(0,0);
  lcd.print("CONTINUITY");

  if (connected == LOW)
  {
    lcd.setCursor(0,1);
    lcd.print("CONNECTED");
    tone(10, 3000);
  }
  if (connected == HIGH) {
    noTone(10);
  }
}
