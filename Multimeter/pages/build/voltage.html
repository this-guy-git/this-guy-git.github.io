Voltage Measurement

What Is Voltage?

Voltage is the electrical pressure that pushes current through a circuit. It tells you how much energy each electron carries as it moves. Measuring voltage is one of the main ways to check how a circuit is working.

How It’s Used

  • Helps you see how much power is available in a battery or supply.
  • Shows if components are receiving the correct voltage.
  • Used in combination with resistors and other parts to calculate current or test circuit behavior.

Testing

Upload this code to test the Voltmeter. 
#include <LiquidCrystal.h>

LiquidCrystal lcd(7, 6, 5, 4, 3, 2);

void setup() {
  lcd.begin(16, 2);
}

void loop() {
  int voltageraw = analogRead(A0);
  float vlt = voltageraw * (5.0 / 1023.0) * (118.0 / 18.0) + 1.0;
  lcd.clear();
  lcd.setCursor(0,0);
  lcd.print("VOLTAGE:");
  lcd.setCursor(0,1);
  if (vlt < 31.0) lcd.print((int)vlt + "V");
  else lcd.print("TOO HIGH");
  delay(1000);
}
Then you can touch the leads to any battery or the arduino's 5v and ground line, the value should show on the LCD.

⚠️ Warning: Do not try to measure voltage with reverse polarity (Leads in opposite directions). This can damage the Arduino and components.
