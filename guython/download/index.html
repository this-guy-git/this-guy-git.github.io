Installation Instructions

Windows Installation

Download and run the installer, or install via command line: 

# Download and install Guython (CMD Prompt)
powershell -Command "$r=Invoke-RestMethod -Uri 'https://api.github.com/repos/this-guy-git/Guython/releases/latest' -Headers @{ 'User-Agent'='PS' }; $exe=$r.assets | Where-Object { $_.name -like '*.exe' }; Invoke-WebRequest $exe.browser_download_url -OutFile $exe.name; Start-Process $exe.name"

Package Manager (GPD)

After installing Guython, you can use GPD to manage packages: 

# Install a package
gpd install package-name

# List installed packages
gpd list

# Check packages from repo
gpd pkgs
